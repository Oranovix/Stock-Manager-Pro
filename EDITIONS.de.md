# Stock Manager Pro — Editionen

Stock Manager Pro ist in zwei Editionen erhältlich.

## Editionsvergleich

| Funktion | **Free** | **Pro** |
|---|---|---|
| Kern-Inventar, Matrix-Ansicht, Kategorien | ✅ | ✅ |
| Barcode-Erstellung + USB-Scanner + Quick Scan | ✅ | ✅ |
| Verkauf / POS mit PDF-Belegen | ✅ | ✅ |
| PDF-Berichte | 3 Kern (Inventar, niedriger Bestand, Transaktionen) | **Alle 14** |
| PCs / Cloud-Synchronisierung | 1 PC, nur lokal | **Mehrere PCs, geteilte Cloud** |
| Telefone / IMEI-Modul | — | ✅ |
| Bestellungen, Preislisten, Inventuren, Lieferanten | Nur-Lese nach 14-tägiger Testphase | ✅ |
| Automatische Updates | Sicherheitskorrekturen | Alle Versionen |
| Support | Community | Direkt |

## Zusagen (jede Edition, für immer)

- Ihre **Daten werden nie als Geisel gehalten**: Export (CSV/JSON), Sicherungen und
  Lesezugriff funktionieren in jeder Edition, lizenziert oder nicht.
- Eine abgelaufene Lizenz deaktiviert die App nie — nur künftige Updates und Pro-
  Module. Sie arbeiten weiter mit der letzten berechtigten Version und all Ihren Daten.
- Keine Telemetrie in irgendeiner Edition.

## Eine Pro-Lizenz erhalten

Kontaktieren Sie **[Oranovix](https://github.com/Oranovix)** für einen Lizenzschlüssel.
Die Aktivierung erfolgt vollständig offline — kein Konto, keine Heimtelefonie.
