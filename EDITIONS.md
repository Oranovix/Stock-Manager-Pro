# Stock Manager Pro — Editions

Stock Manager Pro is available in two editions.

## Edition comparison

| Capability | **Free** | **Pro** |
|---|---|---|
| Core inventory, matrix view, categories | ✅ | ✅ |
| Barcode generation + USB scanner + Quick Scan | ✅ | ✅ |
| Sales / POS with PDF receipts | ✅ | ✅ |
| PDF reports | 3 core (inventory, low stock, transactions) | **All 14** |
| PCs / cloud sync | 1 PC, local only | **Multi-PC shared cloud** |
| Phones / IMEI module | — | ✅ |
| Purchase orders, price lists, audits, suppliers | View-only after 14-day trial | ✅ |
| Auto-updates | Security fixes | All releases |
| Support | Community | Direct |

## Commitments (every edition, forever)

- Your **data is never held hostage**: export (CSV/JSON), backups, and read
  access work in every edition, licensed or not.
- A lapsed license never disables the app — only future updates and Pro
  modules. You keep working on the last entitled version with all your data.
- No telemetry in any edition.
